# Movie Ticket Booking System

Movie Ticket Booking System with the following components:

1. Backend API (Node.js + Express + MongoDB)
2. Frontend Website (React.js)
3. Mobile App (React Native)
4. Admin Panel (React.js)

## Project Structure

- `/backend` - Node.js backend with Express
- `/frontend` - React.js website
- `/mobile` - React Native mobile app
- `/admin` - Admin panel (React.js)

## System Features

The system includes:
- Multi-channel booking (Website, Mobile App, Chatbot)
- Movie management
- Showtime scheduling
- Theater management with seat customization
- Seat selection and booking
- Payment processing
- User management
- Admin dashboard
